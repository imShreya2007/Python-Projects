# Social Bio Generator

A **Python console application** that generates creative and stylish bios for your social media accounts. Users can choose from multiple styles, include emojis, and save their bios as a text file.

## Features
- Interactive input for **name, profession, passion, emoji, and website/handle**.  
- Offers **three stylish bio formats**: Simple Line, Emoji Sandwich, and Vertical Flair.  
- Generates a **ready-to-use bio** for social media profiles.  
- Optionally saves the bio as a **text file**.  
- User-friendly console interface with input validation.  

---

## Usage
- Run the program:
```bash
 python bio_generator.py
```
- Enter your name, profession, passion, favorite emoji, and website/handle.
- Choose a bio style (1, 2, or 3).
- The program generates your bio in the chosen style.
- Optionally, save it as a text file with an auto-generated filename.
