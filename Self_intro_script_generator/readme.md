# Personalized Self Introduction Script Generator

A **Python console application** that generates a friendly, personalized self-introduction script based on user inputs. Ideal for practice sessions, interviews, presentations, or casual introductions.

## Features
- Collects personal details such as **name, age, city, profession, and hobby**.
- Generates a **natural, human-like introduction script**.
- Adjusts wording automatically if the user is a **student**.
- Displays the **current date and time** of generation.
- Simple, readable, and beginner-friendly code structure.

---

## Usage

1. Run the program:
```bash
python self_intro_generator.py
```
2. Enter the requested details when prompted:
Name
- Age
- City
- Profession / Field of study
- Hobby
3. The program prints a custom self-introduction script to the console.

---

## How It Works
- User inputs are collected using standard input.
- A function generates a formatted introduction using conditional logic.
- The datetime module logs the current date and time.
- If the profession is "Student", the script adapts automatically.
